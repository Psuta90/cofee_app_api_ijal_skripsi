## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## migrate

```bash
# unit tests
$ npx prisma migrate dev

# e2e tests
$ npx prisma db seed

```
